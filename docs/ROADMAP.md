# Roadmap de Escalamiento de Perfil (Mateo)

## Objetivo
Conseguir entrevistas y contratación en roles de ingeniería mecatrónica, automatización industrial o IA aplicada en 90 días.

## Portafolio (fase 1)
1. Dashboard KPI de mantenimiento (React + datos CSV)
2. Simulador de línea con lógica tipo PLC (Python)
3. Visión artificial para inspección básica (Python + OpenCV)
4. Agente IA para soporte técnico industrial (LLM + prompts)
5. CV web interactivo (este repositorio)

## Certificados gratuitos (fase 1)
- SQL básico aplicado
- Python aplicado a datos
- Fundamentos de IA/ML
- Automatización/PLC (ruta inicial)

## Regla de calidad por proyecto
- README con: problema, solución, stack, cómo correr, resultados
- Capturas en `public/img/`
- Demo o evidencia ejecutable
